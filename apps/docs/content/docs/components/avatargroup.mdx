## Features

- Stacked (overlapping) or Grid layouts
- Overflow handling (`+3 more`)
- Hover effects for stacked items (expand on hover)
- Integrated with `Avatar` component for consistent styling
- fully accessible `role="group"`

## Props

| Prop | Type | Default | Description |
|:-----|:-----|:--------|:------------|
| `users` | `UserData[]` | `[]` | Array of user objects |
| `size` | `'sm' \| 'md' \| 'lg' \| 'xl'` | `'md'` | Size of avatars |
| `maxDisplay` | `number` | `5` | Max visible before overflow |
| `layout` | `'stack' \| 'grid'` | `'stack'` | Layout mode |
| `spacing` | `'tight' \| 'normal' \| 'loose' \| number` | `'normal'` | Negative margin for stack |
| `renderOverflow` | `(count) => ReactNode` | — | Custom overflow indicator |
| `onAvatarClick` | `(user, index) => void` | — | Click handler |

### UserData Interface

```tsx
interface UserData {
  id?: string | number;
  src?: string;
  username?: string;
  alt?: string;
  fallback?: string;
  isOnline?: boolean;
}
```
