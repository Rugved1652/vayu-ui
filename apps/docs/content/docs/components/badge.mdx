## Features

- Multiple semantic variants
- 3 sizes (`sm`, `md`, `lg`)
- Interactive mode with hover/active states
- Full `HTMLAttributes<HTMLSpanElement>` support
- Flex layout for easy icon composition

## Props

| Prop | Type | Default | Description |
|:-----|:-----|:--------|:------------|
| `variant` | `'primary' \| 'neutral' \| 'outline' \| 'destructive' \| 'success' \| 'warning' \| 'info'` | `'primary'` | Visual style |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Size |
| `interactive` | `boolean` | `false` | Enables hover/click styles |
