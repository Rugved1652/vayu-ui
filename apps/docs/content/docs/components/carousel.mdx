## Features

- **Transition Effects** — `slide`, `fade`, `zoom`, `flip`
- **Bullet Styles** — `dots`, `lines`, `numbers`, `thumbnails`
- **Variants** — `default`, `gallery`, `testimonial`, `product`
- **Autoplay** — With pause on hover and play/pause button
- **Drag / Swipe** — Mouse and touch support
- **Zoom** — Manual zoom controls and zoom-on-hover mode
- **Fullscreen** — Portal-rendered fullscreen modal with keyboard navigation

## Props

### Carousel

| Prop               | Type                          | Default     | Description                              |
| :----------------- | :---------------------------- | :---------- | :--------------------------------------- |
| `items`            | `CarouselItem[]`              | —           | Array of slide items.                    |
| `variant`          | `default \| gallery \| testimonial \| product` | `default` | Visual variant.              |
| `bulletStyle`      | `dots \| lines \| numbers \| thumbnails` | `dots` | Bullet navigation style.            |
| `transitionEffect` | `slide \| fade \| zoom \| flip` | `slide`   | Slide transition effect.                 |
| `autoplay`         | `boolean`                     | `false`     | Enable autoplay.                         |
| `autoplayDelay`    | `number`                      | `3000`      | Autoplay interval (ms).                  |
| `showControls`     | `boolean`                     | `true`      | Show prev/next buttons.                  |
| `showBullets`      | `boolean`                     | `true`      | Show bullet navigation.                  |
| `loop`             | `boolean`                     | `true`      | Loop slides.                             |
| `draggable`        | `boolean`                     | `true`      | Enable drag/swipe.                       |
| `enableZoom`       | `boolean`                     | `false`     | Show zoom controls (gallery variant).    |
| `zoomOnHover`      | `boolean`                     | `false`     | Zoom on mouse hover.                     |
| `maxZoom`          | `number`                      | `3`         | Maximum zoom level.                      |
| `aspectRatio`      | `string`                      | `"16/9"`    | CSS aspect ratio.                        |
| `onSlideChange`    | `(index: number) => void`     | —           | Callback on slide change.                |

### CarouselItem

| Property      | Type         | Description                       |
| :------------ | :----------- | :-------------------------------- |
| `id`          | `string \| number` | Unique identifier.           |
| `image`       | `string`     | Image URL.                        |
| `title`       | `string`     | Slide title.                      |
| `description` | `string`     | Slide description text.           |
| `content`     | `ReactNode`  | Custom content (overrides title/description). |
| `thumbnail`   | `string`     | Thumbnail URL (for thumbnail bullets). |
