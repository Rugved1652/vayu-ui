## Variants

### Default
Transparent background with subtle hover state.

### Bordered
Card-like styling with border and background.

### Filled
Shaded background for visual grouping.

## Collapsible.Group

Wrapping multiple `Collapsible` items in `Collapsible.Group` enables accordion behavior — only one item open at a time by default. Set `allowMultiple` to let multiple items stay open.

## Props

### Collapsible

| Prop               | Type                             | Default     | Description                              |
| :----------------- | :------------------------------- | :---------- | :--------------------------------------- |
| `trigger`          | `ReactNode`                      | —           | Content rendered in the trigger button.  |
| `children`         | `ReactNode`                      | —           | Collapsible content.                     |
| `defaultOpen`      | `boolean`                        | `false`     | Initially open (uncontrolled).           |
| `open`             | `boolean`                        | —           | Controlled open state.                   |
| `onOpenChange`     | `(open: boolean) => void`        | —           | Callback when open state changes.        |
| `variant`          | `default \| bordered \| filled`  | `default`   | Visual variant.                          |
| `size`             | `sm \| md \| lg`                 | `md`        | Size of trigger and content.             |
| `disabled`         | `boolean`                        | `false`     | Disable toggling.                        |
| `icon`             | `ReactNode`                      | auto        | Custom icon (replaces default chevron).  |
| `iconPosition`     | `left \| right`                  | `right`     | Icon placement.                          |
| `triggerClassName`  | `string`                        | —           | Extra classes for the trigger.           |
| `contentClassName`  | `string`                        | —           | Extra classes for the content.           |
| `animationDuration` | `number`                        | `300`       | Height animation duration (ms).          |

### Collapsible.Group

| Prop              | Type        | Default | Description                                  |
| :---------------- | :---------- | :------ | :------------------------------------------- |
| `children`        | `ReactNode` | —       | `Collapsible` items.                         |
| `allowMultiple`   | `boolean`   | `false` | Allow multiple items open simultaneously.    |
| `defaultOpenItems`| `number[]`  | `[]`    | Indices of initially open items.             |
