## Props

### Divider (Root)
| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | Orientation of the divider. |
| `spacing` | `'none' \| 'sm' \| 'md' \| 'lg' \| 'xl' \| '2xl'` | `'md'` | Spacing margin around the divider. |
| `children` | `ReactNode` | - | If provided, used as content (flex container). If not, renders a default line. |

### Divider.Line
| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `variant` | `'solid' \| 'dashed' \| 'dotted' \| 'gradient'` | `'solid'` | Visual style of the line. |
| `color` | `'neutral' \| 'primary' \| ...` | `'neutral'` | Theme color of the line. |
| `size` | `'thin' \| 'normal' \| 'thick' \| 'bold'` | `'normal'` | Thickness preset. |
| `thickness` | `number` | - | Custom pixel thickness. |

### Divider.Label
| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `color` | `'neutral' \| 'primary' \| ...` | `'neutral'` | Text color of the label. |
| `children` | `ReactNode` | - | Label text or content. |
