## Features

- Fixed to top-center with glassmorphism backdrop
- `<nav>` element with `aria-label`
- Tooltips with `role="tooltip"`, `useId`, and `aria-describedby`
- Focus & hover trigger tooltips for keyboard access
- `role="separator"` on dividers
- Configurable logo with `logoText` prop
- Custom link component (e.g. Next.js `Link`)
- `forwardRef` for ref forwarding

## DockItem

| Property    | Type                 | Description                                     |
| :---------- | :------------------- | :----------------------------------------------- |
| `icon`      | `ComponentType`      | Icon component (e.g. Lucide)                     |
| `label`     | `string`             | Tooltip text. Use `"divider"` or `"logo"` for special items |
| `href`      | `string`             | URL — renders as link                            |
| `onClick`   | `() => void`         | Click handler — renders as button                |
| `logoText`  | `string`             | Logo text (only when `label === "logo"`)          |

## FloatingDock Props

| Prop             | Type          | Default           | Description                   |
| :--------------- | :------------ | :---------------- | :---------------------------- |
| `items`          | `DockItem[]`  | —                 | Items to render               |
| `aria-label`     | `string`      | `"Floating dock"` | Accessible name for `<nav>`   |
| `linkComponent`  | `ElementType` | `"a"`             | Custom link component         |
| `className`      | `string`      | —                 | Class on the nav wrapper      |
