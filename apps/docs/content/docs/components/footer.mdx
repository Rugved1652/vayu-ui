## Props

### Footer (Root)

The main wrapper component.

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| children | `ReactNode` | - | The content of the footer. |
| variant | `"default" \| "minimal" \| "centered"` | `"default"` | The layout variant. Applied via data attributes for server rendering. |

### Footer.Section

A section within the footer grid.

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| title | `string` | - | Optional title for the section. |
| children | `ReactNode` | - | The content of the section. |

### Footer.Link & Footer.SocialLink

Link components with hover effects and accessible focus states.

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| href | `string` | - | The URL to navigate to. |
| external | `boolean` | `false` | Whether to open the link in a new tab. |
| children | `ReactNode` | - | The link text or content. |
