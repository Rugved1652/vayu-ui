## Features

- Portal-rendered with collision-aware side flipping
- Four sides — `top`, `bottom`, `left`, `right`
- Three alignment options — `start`, `center`, `end`
- Directional arrow with proper border clipping
- `aria-haspopup`, `aria-expanded`, `aria-describedby` via `useId`
- Focus & blur triggers for keyboard users
- Escape key dismisses card
- Configurable open and close delays
- Scroll & resize repositioning
- Disabled state
- `forwardRef` for ref forwarding

## Props

| Prop               | Type                             | Default    | Description                  |
| :----------------- | :------------------------------- | :--------- | :--------------------------- |
| `content`          | `ReactNode`                      | —          | Card content                 |
| `side`             | `top \| bottom \| left \| right` | `"bottom"` | Preferred side               |
| `align`            | `start \| center \| end`         | `"center"` | Alignment along the edge     |
| `sideOffset`       | `number`                         | `8`        | Gap between trigger and card |
| `alignOffset`      | `number`                         | `0`        | Alignment shift (px)         |
| `openDelay`        | `number`                         | `200`      | Open delay (ms)              |
| `closeDelay`       | `number`                         | `300`      | Close delay (ms)             |
| `contentClassName` | `string`                         | —          | Class on card container      |
| `disabled`         | `boolean`                        | `false`    | Disable hover card           |
| `showArrow`        | `boolean`                        | `true`     | Show directional arrow       |
| `className`        | `string`                         | —          | Class on trigger wrapper     |
