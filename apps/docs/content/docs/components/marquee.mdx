## Anatomy

```tsx
<Marquee speed="normal" direction="left" pauseOnHover>
  <MarqueeContent>
    <MarqueeItem>
      Content...
    </MarqueeItem>
  </MarqueeContent>
</Marquee>
```

## Props

### Marquee

| Prop           | Type                       | Default  | Description                                      |
| :------------- | :------------------------- | :------- | :----------------------------------------------- |
| `speed`        | `"slow" | "normal" | "fast"` | `"normal"` | Speed of the animation.                           |
| `direction`    | `"left" | "right"`         | `"left"` | Direction of the animation.                       |
| `pauseOnHover` | `boolean`                  | `true`   | Whether to pause the animation when hovering.    |
| `gradient`     | `boolean`                  | `true`   | Whether to show fade gradient on edges.          |
