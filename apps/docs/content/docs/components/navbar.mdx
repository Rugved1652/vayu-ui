## Accessibility (WCAG 2.2 AA)

- `aria-label="Main navigation"` on `<nav>`
- `aria-expanded` and `aria-controls` on hamburger toggle
- Focus trap inside mobile menu panel
- Escape key closes mobile menu
- `aria-current="page"` on active items
- All interactive elements have visible focus rings (`focus-visible:ring-2`)
- Body scroll lock when mobile menu is open

## Compound Components

| Component | Description |
|:----------|:------------|
| `Navbar` | Root `<nav>` element with context provider |
| `Navbar.Container` | Max-width + padding wrapper |
| `Navbar.Brand` | Logo / brand slot |
| `Navbar.Items` | Desktop link group (`role="menubar"`) |
| `Navbar.Item` | Individual nav link |
| `Navbar.Actions` | Right-side slot (buttons, avatars) |
| `Navbar.Toggle` | Mobile hamburger button |
| `Navbar.MobileMenu` | Slide-in panel with backdrop |
| `Navbar.MobileItem` | Mobile nav link |
| `Navbar.Separator` | Divider inside mobile menu |

## Navbar Props

| Prop | Type | Default | Description |
|:-----|:-----|:--------|:------------|
| `variant` | `'default' \| 'floating' \| 'bordered'` | `'default'` | Visual style |
| `sticky` | `boolean` | `false` | Stick to top of viewport |

## Container Props

| Prop | Type | Default | Description |
|:-----|:-----|:--------|:------------|
| `maxWidth` | `'sm' \| 'md' \| 'lg' \| 'xl' \| '2xl' \| 'full'` | `'xl'` | Max container width |

## Item Props

| Prop | Type | Default | Description |
|:-----|:-----|:--------|:------------|
| `active` | `boolean` | `false` | Highlights as current page |
