## Features

- Pure TypeScript implementation (no external QR libraries)
- Supports logos/images in the center (with optional excavation)
- Customizable colors (foreground and background)
- Configurable error correction levels (L, M, Q, H)
- Responsive and accessible (`role="img"`, `aria-label`)
- SVG-based rendering for crisp scaling

## Props

| Prop | Type | Default | Description |
|:-----|:-----|:--------|:------------|
| `value` | `string` | — | The data to encode |
| `size` | `number` | `200` | Size in pixels |
| `level` | `'L' \| 'M' \| 'Q' \| 'H'` | `'M'` | Error correction level |
| `bgColor` | `string` | `'white'` | Background color |
| `fgColor` | `string` | `'black'` | Module color |
| `includeMargin` | `boolean` | `true` | Add quiet zone |
| `imageSettings` | `QRCodeImageSettings` | — | Center logo config |
| `className` | `string` | — | Class for wrapper div |

### QRCodeImageSettings

```tsx
interface QRCodeImageSettings {
  src: string;      // Image URL
  width: number;    // Width in pixels
  height: number;   // Height in pixels
  excavate?: boolean; // Clear modules behind image
}
```
