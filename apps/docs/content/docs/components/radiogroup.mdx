## Variants

### Default
Standard radio buttons with a circular indicator.

### Card
Card-style radio options with optional icon and description — great for feature selection.

### Button
Compact pill-style buttons for inline selections like sizes or categories.

## Props

### RadioGroup

| Prop          | Type                          | Default      | Description                               |
| :------------ | :---------------------------- | :----------- | :---------------------------------------- |
| `value`       | `string`                      | —            | Controlled selected value.                |
| `defaultValue`| `string`                      | `""`         | Uncontrolled default value.               |
| `onChange`     | `(value: string) => void`     | —            | Callback when selection changes.          |
| `name`        | `string`                      | auto         | HTML name attribute for the radio group.  |
| `disabled`    | `boolean`                     | `false`      | Disable all items.                        |
| `size`        | `sm \| md \| lg`              | `md`         | Size of radio items.                      |
| `variant`     | `default \| card \| button`   | `default`    | Visual variant.                           |
| `color`       | `primary \| secondary \| success \| warning \| error \| info` | `primary` | Color theme. |
| `orientation` | `vertical \| horizontal`      | `vertical`   | Layout direction.                         |
| `label`       | `string`                      | —            | Group label.                              |
| `description` | `string`                      | —            | Helper text below label.                  |
| `error`       | `boolean`                     | `false`      | Show error styling.                       |
| `errorText`   | `string`                      | —            | Error message text.                       |
| `required`    | `boolean`                     | `false`      | Mark as required.                         |

### RadioGroup.Item

| Prop          | Type              | Default | Description                           |
| :------------ | :---------------- | :------ | :------------------------------------ |
| `value`       | `string`          | —       | Value for this radio option.          |
| `label`       | `string`          | —       | Label text.                           |
| `description` | `string`          | —       | Description text (default & card).    |
| `disabled`    | `boolean`         | `false` | Disable this specific item.           |
| `icon`        | `ReactNode`       | —       | Icon element (card & button variants).|
