## Features

- Compound pattern — compose `Panel` and `Handle` sub-components
- Horizontal and vertical layouts
- Percentage-based sizing with min / max constraints
- Mouse and touch drag support (with body cursor lock)
- `role="separator"` with `aria-valuenow` / `aria-valuemin` / `aria-valuemax`
- Keyboard: Arrow keys (2% step), Shift+Arrow (10% step), Home / End (collapse / expand)
- Visual grip dots on handles
- Multiple panels (3+) with independent handles
- `forwardRef` on all sub-components

## Sub-components

| Component | Description |
|:----------|:------------|
| `ResizablePane` | Root — flex container + context provider |
| `ResizablePane.Panel` | Sized panel section |
| `ResizablePane.Handle` | Draggable separator between panels |

## ResizablePane Props

| Prop        | Type                         | Default        | Description     |
| :---------- | :--------------------------- | :------------- | :-------------- |
| `direction` | `horizontal \| vertical`     | `"horizontal"` | Layout direction |

## Panel Props

| Prop          | Type     | Default | Description            |
| :------------ | :------- | :------ | :--------------------- |
| `defaultSize` | `number` | `50`    | Initial size (%)       |
| `minSize`     | `number` | `10`    | Minimum size (%)       |
| `maxSize`     | `number` | `90`    | Maximum size (%)       |

## Handle Props

| Prop         | Type     | Default | Description                      |
| :----------- | :------- | :------ | :------------------------------- |
| `step`       | `number` | `2`     | Keyboard step size (%)           |
| `aria-label` | `string` | Auto    | Accessible label for the handle  |

## Keyboard Shortcuts

| Key              | Action               |
| :--------------- | :------------------- |
| `← ↑`            | Shrink panel before  |
| `→ ↓`            | Expand panel before  |
| `Shift + Arrow`  | 5× bigger step       |
| `Home`           | Collapse to min      |
| `End`            | Expand to max        |
