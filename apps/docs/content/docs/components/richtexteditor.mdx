## Features

- **WYSIWYG Toolbar**: Bold, Italic, Underline, Strikethrough, Inline Code, Bullet/Numbered Lists, Image
- **Markdown Shortcuts**: Type `#` for H1, `##` for H2, `**text**` for bold, `>` for blockquote, `-` or `*` for lists
- **@Mentions**: Type `@` to search and insert mentions with keyboard navigation
- **Images**: Upload from file or insert via URL
- **Output**: Markdown string, HTML string, or raw EditorState
- **Built on Lexical**: Meta's extensible text editor framework

## Props

| Prop | Type | Default | Description |
|:-----|:-----|:--------|:------------|
| `defaultValue` | `string` | ‚Äî | Initial content as markdown |
| `placeholder` | `string` | `'Start writing‚Ä¶'` | Placeholder text |
| `onChange` | `(markdown: string) => void` | ‚Äî | Markdown output callback |
| `onHtmlChange` | `(html: string) => void` | ‚Äî | HTML output callback |
| `onEditorStateChange` | `(state: EditorState) => void` | ‚Äî | Raw state callback |
| `mentions` | `MentionUser[]` | ‚Äî | List of mentionable users |
| `onImageUpload` | `(file: File) => Promise<string>` | ‚Äî | Custom image upload handler |
| `disabled` | `boolean` | `false` | Disable editing |
| `minHeight` | `string` | `'200px'` | Minimum editor height |
| `editorRef` | `(editor: LexicalEditor) => void` | ‚Äî | Access the Lexical editor instance |

## Markdown Shortcuts

| Shortcut | Result |
|:---------|:-------|
| `# ` | Heading 1 |
| `## ` | Heading 2 |
| `### ` | Heading 3 |
| `**text**` | **Bold** |
| `*text*` | *Italic* |
| `` `code` `` | `Inline code` |
| `> ` | Blockquote |
| `- ` or `* ` | Bullet list |
| `1. ` | Numbered list |
| `---` | Horizontal rule |
