## Features

- Supports single `value` or multiple (range)
- Keyboard accessible (`Arrow` keys, `Home`, `End`)
- Mouse and Touch dragging support
- Custom `step` interpolation
- Form submission support via hidden inputs

## Props

| Prop | Type | Default | Description |
|:-----|:-----|:--------|:------------|
| `defaultValue` | `number[]` | `[0]` | Initial value(s) |
| `value` | `number[]` | — | Controlled value(s) |
| `min` | `number` | `0` | Minimum value |
| `max` | `number` | `100` | Maximum value |
| `step` | `number` | `1` | Stepping interval |
| `disabled` | `boolean` | `false` | Disable interaction |
| `onValueChange` | `(val: number[]) => void` | — | Event handler for changes |
| `onValueCommit` | `(val: number[]) => void` | — | Event handler for drag/key end |
| `name` | `string` | — | Name for form input |
