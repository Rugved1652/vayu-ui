## Features

- Built on native `input[type="checkbox"]` for full accessibility
- Supports `label` and `description` props with automatic layout
- 3 sizes (`sm`, `md`, `lg`)
- Keyboard navigation & focus rings
- Error state support

## Props

Extends `InputHTMLAttributes<HTMLInputElement>` (except `size`).

| Prop | Type | Default | Description |
|:-----|:-----|:--------|:------------|
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Size |
| `label` | `ReactNode` | — | Primary Label |
| `description` | `ReactNode` | — | Helper text |
| `error` | `boolean` | `false` | Visual error state |
