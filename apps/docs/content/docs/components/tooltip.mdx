## Features

- Portal-rendered with viewport clamping
- Four positions — `top`, `bottom`, `left`, `right`
- Seven colour variants — `default`, `primary`, `secondary`, `success`, `warning`, `error`, `info`
- Directional arrow with variant-matched colour
- `aria-describedby` linking trigger to tooltip (`useId`)
- Focus & blur triggers for keyboard users
- Escape key dismisses tooltip
- Configurable show delay
- Disabled state
- `forwardRef` for ref forwarding

## Props

| Prop        | Type                                                                     | Default     | Description                  |
| :---------- | :----------------------------------------------------------------------- | :---------- | :--------------------------- |
| `content`   | `ReactNode`                                                              | —           | Tooltip content              |
| `position`  | `top \| bottom \| left \| right`                                         | `"top"`     | Placement                    |
| `delay`     | `number`                                                                 | `200`       | Show delay (ms)              |
| `variant`   | `default \| primary \| secondary \| success \| warning \| error \| info` | `"default"` | Colour variant               |
| `disabled`  | `boolean`                                                                | `false`     | Disable tooltip              |
| `showArrow` | `boolean`                                                                | `true`      | Show directional arrow       |
| `className` | `string`                                                                 | —           | Class on trigger wrapper     |
