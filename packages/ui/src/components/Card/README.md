# Card

## Anatomy

- `card.tsx` — Composition: Card root with interactive logic and styling
- `header.tsx` — UI: Card header with title, subtitle, avatar, and action
- `media.tsx` — UI: Card media/image with overlay support
- `content.tsx` — UI: Card content wrapper
- `footer.tsx` — UI: Card footer for actions
- `hooks.ts` — Logic: Keyboard interaction handler
- `types.ts` — Types: TypeScript interfaces
- `index.ts` — Public API: Compound exports

## Use Cases

- Display content with optional media, headers, and footers
- Create clickable cards for navigation or actions
- Build card-based layouts with keyboard accessibility
- Support linked cards with automatic security attributes
