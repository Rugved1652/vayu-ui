# Show

## Anatomy

- show.tsx — conditional rendering based on a truthy condition
- match.tsx — first-match switch/case rendering with Case and Default branches
- types.ts — type definitions
- index.ts — public API

## Use Cases

- Conditionally render UI blocks with optional fallback
- Replace ternary chains with declarative `<Match>` / `<Case>` / `<Default>`
- Render children as a function with access to the truthy value
