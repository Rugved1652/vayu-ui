# AudioPlayer

## Anatomy
- `audioplayer.tsx` — composition (contexts, state, event handlers, prop getters)
- `types.ts` — types
- `hooks.ts` — hooks
- `utils.ts` — utilities
- `index.ts` — export

## Use Cases
- Audio playback with playlist support
- Keyboard controls (Space/k: play/pause, m: mute, arrows: seek/volume)
- WCAG compliant with ARIA labels and keyboard navigation
- Custom audio player UIs via prop getters
