# Carousel

## Anatomy
- `carousel.tsx` — root component with context provider and state management
- `viewport.tsx` — viewport with touch and keyboard navigation
- `slide.tsx` — individual slide wrapper
- `previous.tsx` — previous navigation button
- `next.tsx` — next navigation button
- `bullets.tsx` — slide indicator dots
- `play-pause.tsx` — play/pause slideshow control
- `speed-control.tsx` — playback speed selector
- `gallery.tsx` — thumbnail gallery
- `hooks.ts` — context hook and reduced motion detection
- `types.ts` — TypeScript types
- `index.ts` — public API

## Use Cases
- Image galleries and slideshows
- Product showcases with multiple views
- Auto-playing presentations with manual controls
- Touch-enabled carousels with keyboard navigation
