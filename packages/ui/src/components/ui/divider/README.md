# Divider

## Anatomy
- divider.tsx — composition (root layout + compound assembly)
- divider-line.tsx — UI (line with variant/color/size)
- divider-label.tsx — UI (label text between lines)
- types.ts — types & design token maps
- index.ts — public API

## Use Cases
- Horizontal or vertical visual separation between content sections
- Labeled dividers with text or icons between line segments
- Decorative dividers with customizable color, thickness, and style
