# FloatingDock

## Anatomy
- floatingdock.tsx — composition (root nav wrapper)
- container.tsx — glassmorphism layout with cloneElement injection
- item.tsx — interactive nav item with tooltip
- logo.tsx — brand logo item
- types.ts — types
- index.ts — export

## Use Cases
- Floating navigation bar fixed to the top of a page
- App-level dock with icon-based navigation items
- Brand header with logo and action icons
