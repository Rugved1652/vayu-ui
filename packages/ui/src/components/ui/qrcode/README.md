# QRCode

## Anatomy

- `qrcode.tsx` — Composition: SVG rendering + layout
- `encoder.ts` — Logic: QR code encoding engine (byte mode)
- `types.ts` — Types: shared types
- `index.ts` — Public API: exports

## Use Cases

- Encoding URLs, text, or any string data into scannable QR codes
- Custom-branded QR codes with center logo overlay
- Size and color customization for integration into any layout
