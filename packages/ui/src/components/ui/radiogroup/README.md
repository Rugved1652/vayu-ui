# RadioGroup

## Anatomy
- radiogroup.tsx — composition (UI + wiring)
- radio-item.tsx — UI (presentational radio item)
- hooks.ts — logic (context + useRadioGroup hook)
- types.ts — types (RadioGroupProps, RadioItemProps, RadioGroupContextType)
- index.ts — export

## Use Cases
- Single-select form controls with option labels and descriptions
- Settings panels where users choose one option from a list
- Survey forms with vertical or horizontal radio layouts
