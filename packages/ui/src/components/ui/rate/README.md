# Rate

## Anatomy
- rate.tsx — composition (context provider + state + wiring)
- stars.tsx — UI: stars container + individual star rendering
- label.tsx — UI: label
- description.tsx — UI: description
- value.tsx — UI: value display + text label
- container.tsx — UI: layout container
- error-text.tsx — UI: error message
- hooks.ts — logic (context + useRate hook)
- types.ts — types
- index.ts — public API

## Use Cases
- Star rating input with half-star precision
- Read-only rating display
- Custom icon support for rating visualization
- Form field with labels, descriptions, and error states
