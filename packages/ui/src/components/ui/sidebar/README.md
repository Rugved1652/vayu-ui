# Sidebar

## Anatomy
- sidebar.tsx — composition (provider + sidebar shell)
- sidebar-header.tsx — UI: header section
- sidebar-content.tsx — UI: scrollable content area
- sidebar-footer.tsx — UI: footer section
- sidebar-menu.tsx — UI: menu nav + grouped menu
- sidebar-menu-item.tsx — UI: menu item with icon, badge, tooltip, subitems
- sidebar-toggle.tsx — UI: collapse/expand button
- mobile-menu-button.tsx — UI: mobile hamburger/close button
- hooks.ts — SidebarContext + useSidebar hook
- types.ts — all prop interfaces
- index.ts — public API

## Use Cases
- Responsive sidebar navigation with collapsible state
- Nested menu items with expandable submenus
- Mobile-first overlay navigation with focus trap
